# EQUIPMENT-TRACKER

How to Run the Project

This project has two parts: 
1. backend 
2. frontend.
--------------------------------------------------------------

What You Need: 
Make sure Node.js is installed on your system.
Ensure that the Node.js is installed on your by running this in command prompt:
1. node -v
2. npm -v
If these commands show version numbers, you’re good to go.
---------------------------------------------------------------

Starting the Backend: 

1. Open a terminal.

2. Go into the backend folder:
cd backend

3. Install the required packages:
npm install

4. Start the backend server:
npm start

Once it starts, you should see a message saying that the backend is running on port 5000.

-------------------------------------------------------------------

Starting the Frontend: 

1. Open a new terminal window.

2. Go into the frontend folder:
cd frontend

3. Install the frontend dependencies:
npm install

4. Start the frontend:
npm run dev

After this, you’ll see a local URL (usually http://localhost:5173
).
Open this link in your browser to use the app.

-------------------------------------------------------

Using the Application: 

1. You can add new equipment using the form at the top.
2. Existing equipment will appear in the table below.
3. Use the Edit button to update details or the Delete button to remove an item.
4. All changes are saved through the backend.

--------------------------------------------------------

Notes: 

1. The backend should be running before you start the frontend.
2. To stop either server, press Ctrl + C in the terminal.
